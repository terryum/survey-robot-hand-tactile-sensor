# Contributing

Thank you for your interest in improving this book! We welcome contributions from the tactile sensing and robotics community.

## How to Contribute

This project uses an **issue-based contribution workflow**. Rather than submitting pull requests directly, contributors open GitHub Issues with their suggestions, references, or corrections. The author reviews each submission and integrates approved content into the book.

### 1. Report an Error

Found a typo, factual error, or broken link?

- Use the [**Report Error**](https://github.com/terryum/book-robot-hand-tactile-sensor/issues/new?template=error-report.yml) template
- Include the chapter, section, and specific text
- Suggest a correction if possible

### 2. Suggest New Content

Have a paper, product, dataset, or topic that should be covered?

- Use the [**Suggest Content**](https://github.com/terryum/book-robot-hand-tactile-sensor/issues/new?template=suggest-content.yml) template
- Describe what should be added and why
- Provide reference links (papers, product pages, Notion pages, blog posts, etc.)
- AI will help digest your materials and draft content for the author's review

### 3. Improve Translation

Notice an awkward or inaccurate translation in the Korean or English version?

- Use the [**Translation Improvement**](https://github.com/terryum/book-robot-hand-tactile-sensor/issues/new?template=translation-fix.yml) template
- Provide the current text and your suggested improvement

## Contribution Workflow

```
You: Open an Issue (using a template)
  → Author: Reviews the issue and labels it
  → Author: Integrates content with AI assistance
  → Author: Publishes the update (book + website + PDF)
  → Your name is added to the Contributors table in README
  → The issue is closed with a link to the commit
```

## Credit Policy

Every accepted contribution is recognized in two ways:

1. **README Contributors table** — Your GitHub avatar, username, and contribution details are listed, sorted by contribution amount
2. **Chapter references** — When content is based on your materials, you are credited in the chapter's reference list (e.g., `Choi, S. (Korea University). (2026). Commercial tactile sensor survey.`)

Contribution types:
- :book: Content (new sections, product info, practical experience)
- :bug: Error fix (typos, factual corrections, broken links)
- :globe_with_meridians: Translation improvement
- :camera: Figures (photos, diagrams, illustrations)
- :books: References (papers, citations)
